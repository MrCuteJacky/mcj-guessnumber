## GuessNumber
「[GuessNumber](https://mrcutejacky.github.io/mcj-guessnumber/)」是使用Angular開發的經典猜數字小遊戲。

### Tools
1. Angular 10.2.0
2. GitHub Pages

### Development environment
```console
npm install
```

### Run in local
```console
ng serve
```

### Build Production
```console
ng build --prod --output-path docs --base-href
```
